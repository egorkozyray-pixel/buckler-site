# Buckler Site — Deploy Package

Static HTML site. No build step. Drop into any static host (Netlify / Vercel / GitHub Pages / Nginx / S3+CloudFront / Cloudflare Pages).

## Structure

- `index.html` — entry / chooser page (Professional vs Digital)
- `Buckler.html` — Buckler Professional landing
- `Buckler Digital.html` — Buckler Digital landing
- `assets/` — product images, logo

## Deploy

Just upload the entire folder. No server-side code, no env vars, no dependencies — pure HTML/CSS/JS.

Fonts (Syne, DM Sans, JetBrains Mono) are loaded from Google Fonts CDN.
